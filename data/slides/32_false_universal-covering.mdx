# Universal cover

$\widetilde{X}=\widetilde{X}(G, Y, T)$
<br />

$\pi=\pi_1(G, Y, T)$ acts on $\widetilde{X}$
<br />

$p:\widetilde{X}\to Y$ inducing an isomorphism $\pi\backslash\widetilde{X}\cong Y$ + sections
<br />

$\pi_P=Stab(\widetilde{P})$ is equal to $G_P$ (for edges $\sigma_\gamma(G_{t(\gamma)}$)

- $V(\widetilde{X})=\bigsqcup_{P\in V(Y)} \pi P=\bigsqcup_{P\in V(Y)}\pi/\pi_P$
<div style={{ height: '-2px' }} />
- $E(\widetilde{X})=\bigsqcup_{\gamma\in E(Y)}\pi \gamma=\bigsqcup_{\gamma\in E(Y)}\pi/\pi_\gamma$
<div style={{ height: '-2px' }} />

Such $\overline{X}$ is a tree whenever $T$ is a maximal tree in $Y$. Trust me.

### Example

If all $G_P=\{1\}$ then $\overline{X}$ is the normal universal covering of $Y$ relative to $T$ -> Cayley graph of $\pi_1(Y)=F_n$.

### Example

If $Y$ is a segment with vertices $P$ and $Q$ connected via an edge $\gamma$ then the universal cover of $Y$ is a tree on which $G_P\ast_{G_\gamma}G_Q$ acts with fundamental domain equal to $Y$.
