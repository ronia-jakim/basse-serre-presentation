# Tree isometries
<br />

<Theorem title="Twierdzonko">
  Let $T$ be a tree and $\phi:T\to T$ its isometry. Then
  - if $\phi$ is elliptic then $Min(\phi)=Fix(\phi)$ is a subtree or the middle of some edge
  - if $\phi$ is hyperbolic then $Min(\phi)$ is a line $L\subseteq T$.
</Theorem>

The line $L$ in hyperbolic isometry is called *axis* of $\phi$.
